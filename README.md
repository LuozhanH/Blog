# Blog
### 博客(后端)

这里简单说一下博客的基础组成：

前端使用: vue全家桶 + axios + elementui + webpack

后端使用: node + express + mongodb + moogoose

服务端配合nginx配置、pm2跑node服务

有搭建自己博客想法的朋友，可以先学习以上内容，当然也可以选择你喜欢的语言，毕竟条条道路通罗马！
